Fronend: 
 Todo elemento o funcion que puede represant el navegador in necesidad de 
depender de un servidor para llevar a cabo dichas tareas.

Backend:
 Toda consulta que requiera de datos o resoluciones por parte del servidor
es decir, que el navegador no las puede llevar a cabo por su cuenta.

Etiquetas de bloque y etiquetas de linea:
Etiquetas de bloque:
        Aquellas que generan un salto de linea y por defecto ocupan
        todo el ancho del contenedor y se les puede dar tamaño y funcionan
        principalmente como elementos contenedores.
Etiquetas de linea:
        Aquellas que comparten linea con otras etiqueta a diferencia de las de bloque no
        generan salto de line ay su tamaño esta definido por su contenido.
        El ancho de las mismas es automatico y funcionan como elementos de contenido

span vs div:
	El span es para elementos cortos(no se puede colocar un <p>)por ejemplo una palabra o una oracion
	El div es para parrafos largos por ejemplo por ejemplo podria ir un <p>
